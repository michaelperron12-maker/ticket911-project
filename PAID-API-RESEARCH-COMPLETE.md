# 9. KEY LIMITATIONS & GAPS

## Quebec:
- **NO court records API exists** for Quebec municipal/traffic courts
- **SAAQ has no public API** for driver records
- Must rely on CanLII (free) + SOQUIJ (paid) for case law
- Driver records require client self-service through SAAQclic portal
- All major US court record APIs (UniCourt, Trellis, Docket Alarm) have ZERO Quebec coverage

## Ontario:
- **No public API** for Ontario court records (ICON system not developer-accessible)
- Ontario Courts Public Portal (OCPP) exists but no API
- MTO TIPS portal has no developer API
- Certn and Checkr can pull Ontario driver abstracts via MTO
- CanLII provides Ontario case law

## New York:
- **Best covered jurisdiction** - multiple API options
- UniCourt, Trellis, Docket Alarm all cover NY courts
- PACER covers NY federal courts
- NYS DMV DIAL-IN provides direct driver record access at $7/search
- All major legal research platforms cover NY
- Traffic Violations Bureau (TVB) has online lookup but no formal API

## General:
- Most APIs are US-focused; Canadian coverage is significantly weaker
- No single API covers all 3 jurisdictions adequately
- Quebec requires the most manual/workaround approaches
- Building a Quebec-specific scraper/integration may be necessary
