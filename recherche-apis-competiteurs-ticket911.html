2. APIs GRATUITES

Donnees ouvertes accessibles immediatement — le coeur du systeme

New York — Le jackpot des donnees ouvertes

Open Parking & Camera Violations NYC

data.cityofnewyork.us/resource/nc67-uf89.json
GRATUIT API REST NY CRITIQUE

85.4 millions de tickets — la plus grosse base de donnees de tickets au monde. Contient: plaque, amende, statut de paiement, montant du, image du ticket. Mise a jour hebdomadaire.

Exemple : ?plate=ABC1234&state=NY retourne tous les tickets d'une plaque en < 1 seconde.

Moving Violations NYPD (B Summons)

data.cityofnewyork.us/resource/bme5-7ty4.json
GRATUIT API REST NY

5 millions+ de contraventions de mouvement (code VTL, date, coordonnees GPS, precinct, plaque). Donnees NYPD mandatees par la loi.

Traffic Tickets NYS Statewide

data.ny.gov/resource/q4hy-kbtf.json
GRATUIT API REST NY CRITIQUE

10.67 millions de tickets pour TOUT l'etat de New York. Code violation, description, cour, agence de police, age, genre. Parfait pour analytics et strategies de defense.

NYS Vehicle & Traffic Law — Texte complet

legislation.nysenate.gov/api/3/laws/VAT
GRATUIT API REST NY CRITIQUE

Le texte COMPLET de chaque article du Vehicle & Traffic Law en JSON. Peut recuperer n'importe quelle section (ex: VTL 1180 — vitesse). Versions historiques disponibles par date. Open source.

Autres datasets NYC gratuits

GRATUIT NY

• Speed Camera Violations (43 champs, mensuel) — hekt-pqw7
• DOF Parking Violation Codes (97 codes, table de reference) — ncbg-6agr
• NYPD Collisions (tous accidents reportes) — h9gi-nx95
• NYC DOT Speed Data (temps reel) — linkdata.nyctmc.org
• 511NY API (traffic temps reel tout NYS) — 511ny.org/developers

Note technique : Tous les datasets NYC/NYS utilisent la plateforme Socrata (SODA API). Langage de requete SoQL (comme SQL). Library Python : pip install sodapy. App token gratuit = 1,000 requetes/heure.

Quebec

CanLII API — Jurisprudence canadienne

api.canlii.org/v1/
GRATUIT API REST QC ON CRITIQUE

LA source #1 pour le Canada. Acces a toute la jurisprudence traffic du Quebec et de l'Ontario. Texte du Code de la securite routiere (C-24.2), Highway Traffic Act. Citateur (cas cites/citant). API key gratuite sur demande.

Libraries: canliicalls (Python/PyPI), canlii-api (npm), pycanlii (GitHub)

Donnees Quebec SAAQ (CKAN API)

donneesquebec.ca/recherche/api/3/action/
GRATUIT API REST QC

8 datasets SAAQ : Constats d'infraction, rapports d'accident, vehicules en circulation, permis de conduire, non-conformites, interventions, indemnisation. Formats CSV. Requetes SQL directes sur les donnees.

SAAQ Points d'inaptitude & Code C-24.2

GRATUIT PDF A PARSER QC

Pas d'API. Disponible en PDF sur saaq.gouv.qc.ca — il faut parser les PDFs pour construire une base structuree des infractions, points et amendes. Le texte du Code C-24.2 est aussi dispo via CanLII (mieux).

Lois federales Canada (XML/GitHub)

github.com/justicecanada/laws-lois-xml
GRATUIT QC ON

XML complet de toutes les lois federales. Utile pour le Code criminel (conduite avec facultes affaiblies, conduite dangereuse).

Ontario

Schedule 43 — Set Fines (558+ infractions HTA)

ontariocourts.ca/ocj/provincial-offences/set-fines/schedule-43/
GRATUIT SCRAPING HTML ON

Table HTML propre avec 558+ infractions HTA : item, infraction, section HTA, amende fixe. MAJ janvier 2026. Facilement parseable avec BeautifulSoup/cheerio.

A2AJ Canadian Legal Data

api.a2aj.ca/docs | huggingface.co/datasets/a2aj/canadian-case-law
GRATUIT API REST ON

17,000+ decisions Cour d'appel Ontario. Pas de legislation provinciale (federal seulement).

6. Apps MORTES — Lecons critiques

Ce qui a tue les autres — pour eviter les memes erreurs

TIKD (Florida) — MORT

FERME 2022

Ce qui s'est passe : La Cour supreme de Floride a decide 4-3 que TIKD pratiquait ILLEGALEMENT le droit. L'entreprise n'etait pas un cabinet d'avocats mais servait d'intermediaire. TIKD a poursuivi le Barreau pour $20M en antitrust — et a perdu.

Lecon : TOUJOURS operer sous un cabinet d'avocats ou paralegals licencies.

DoNotPay — SANCTIONNE

FTC $193K amende (2025)

Ce qui s'est passe : Se presentait comme "le premier robot avocat au monde". AI entrainee avec GPT-J + IBM Watson. La FTC a determine que les affirmations etaient fausses — l'AI ne fournissait pas vraiment des services juridiques. $193K d'amende + interdit de dire que l'AI remplace un avocat. Tentative d'envoyer l'AI en cour = annulee apres menaces du Barreau.

Lecon : Ne JAMAIS promettre que l'AI remplace un avocat. Disclaimer clair obligatoire.

Fixed (Shark Tank) — MORT

FERME 2016

Ce qui s'est passe : Apparue au Shark Tank (Mark Cuban a offert $700K pour 7% — deal jamais conclu). Acquise par Lawgix (cabinet d'avocats), puis fermee. Probleme: les municipalites ont BLOQUE leurs soumissions automatisees.

Lecon : Les municipalites peuvent bloquer les submissions automatisees. Prevoir des canaux alternatifs.

ROSS Intelligence — MORT

FERME 2021

Ce qui s'est passe : Poursuivi par Thomson Reuters pour violation de droits d'auteur sur les donnees legales de Westlaw.

Lecon : Utiliser les sources de donnees ouvertes (CanLII, government data). Eviter de copier des bases proprietaires.

4 regles d'or extraites des echecs :
1. Operer SOUS un cabinet licencie (pas a cote)
2. L'AI ASSISTE, l'avocat DECIDE (jamais l'inverse)
3. Utiliser seulement des donnees publiques/ouvertes
4. Prevoir que les municipalites/barreaux peuvent etre hostiles
